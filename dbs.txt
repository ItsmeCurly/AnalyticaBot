Messages:
(Stores important data on a message)

id: auto-increment integer
member_id: integer
content: text
channel: integer
guild: integer
time: timestamp

UserProfiles:
(Stores user id and name for cross referencing)

id: auto-increment integer
user_id: integer
user_name: text
avatar_url: text
last_online: timestamp

ServerRef:
(Stores server id and name and all channels on it for cross reference)

id: auto-increment integer
guild: integer
guildname: text
channel: integer
channelname: text
last_updated: timestamp
